# Sign Language Detection

A Machine Learning and Computer Vision project that detects and recognizes sign language gestures in real time. The system helps bridge communication gaps between hearing-impaired individuals and others by converting hand gestures into readable text.


## Features

- Real-time sign language detection
- Hand gesture recognition
- Machine Learning based classification
- User-friendly interface
- Fast and accurate predictions
- Supports multiple sign gestures
- Real-time camera input processing

## Technologies Used

## How It Works

1. Capture hand gestures using a webcam.
2. Preprocess image frames using OpenCV.
3. Extract important hand features.
4. Pass features to the trained ML model.
5. Predict the corresponding sign language gesture.
6. Display the detected character or word.
